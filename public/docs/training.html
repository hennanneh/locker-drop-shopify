Seller Training Guide

Everything you need to know to manage LockerDrop orders

Welcome to LockerDrop!

LockerDrop lets your customers pick up their orders from secure smart lockers instead of waiting at home for delivery. This guide will teach you everything you need to know to manage orders and provide great customer service.

What is LockerDrop?

LockerDrop is a shipping option that lets customers pick up their orders from Harbor smart lockers. Instead of traditional home delivery:

  • Customers choose a convenient locker location at checkout
  • You drop off packages at the locker using a secure access code
  • Customers pick up when it's convenient for them using their own code

Benefits for Your Business

  • No missed deliveries - Customers pick up when they want
  • Reduced shipping costs - Often cheaper than last-mile delivery
  • Faster fulfillment - Drop off locally, no waiting for carriers
  • Happy customers - Flexible pickup on their schedule

Order Lifecycle

Every LockerDrop order goes through these stages:

1

Order Placed

Customer checks out and selects LockerDrop Pickup

2

Pending Drop-off Your action needed

You receive the order with a drop-off access code

3

Dropped Off

You've placed the package in the locker

4

Ready for Pickup Customer notified

Customer receives notification with their pickup code

5

Picked Up Complete!

Customer has retrieved their package

Order Status Colors

Status Color Meaning
Pending Drop-off Orange Waiting for you to drop off
Ready for Pickup Green Package in locker, customer notified
Completed Gray Customer has picked up
Expired Red Customer didn't pick up in time

How to Drop Off a Package

Step 1: View the Order

  1. Go to Orders tab
  2. Find orders with "Pending Drop-off" status
  3. Click "View" to see the order details

Step 2: Note the Drop-Off Information

In the order details, you'll see:

  • Locker Location - Address where to drop off
  • Locker Name - e.g., "Harbor Locker - Downtown Dallas"
  • Drop-off Link - Click to open the locker

Step 3: Prepare the Package

Pack the order securely and make sure it fits the assigned locker size:

Size Dimensions
Small Up to 12" x 8" x 4"
Medium Up to 16" x 12" x 8"
Large Up to 20" x 16" x 12"
X-Large Up to 24" x 20" x 16"

Step 4: Go to the Locker

  1. Travel to the locker location
  2. Find the Harbor Locker kiosk
  3. Click the Drop-off Link on your phone to open the locker

Step 5: Place the Package

  1. The assigned locker door will open
  2. Place the package inside
  3. Close the door securely
  4. The system will confirm the drop-off
Note: Once you complete the drop-off, the customer will automatically receive their pickup notification with a link to open the locker.

Managing Orders

Viewing All Orders

  1. Go to the Orders tab
  2. Use filters to find specific orders:
    • Active - Pending and Ready orders
    • Pending Drop-off - Waiting for you
    • Ready for Pickup - In locker
    • Completed - Picked up
    • All - Everything

Resending Customer Emails

If a customer lost their pickup code:

  1. Find the order
  2. Click "View"
  3. Click "Resend Pickup Email"
  4. Customer will receive a new email with their link

Setting Up Your Lockers

Finding Locker Locations

  1. Go to My Lockers tab
  2. Use the search box to find lockers by city, state, or zip code
  3. Search results show lockers within 50 miles, sorted by distance
  4. Each locker displays the distance (e.g., "2.5 mi")
  5. Click on a location to select/deselect it
  6. Selected lockers will show a checkmark
  7. Click "Save Changes"

Search Tips

  • Search by city - "Dallas" or "Dallas, TX" shows nearby lockers sorted by distance
  • Search by zip code - "75219" finds lockers within 50 miles of that zip
  • Search by state - "Texas" shows all lockers in that state
  • Use "My lockers only" - Filter to see just your selected locations

Tips for Choosing Lockers

  • Choose convenient locations - Near your store or on your regular routes
  • Consider your customers - Select lockers in areas where customers live/work
  • Start small - Begin with 2-3 locations and expand as needed
  • Check hours - Some locations have limited access hours

Setting Product Sizes

LockerDrop automatically selects the right locker size based on your product dimensions. Set these in the Product Sizes tab for accurate sizing.

Why Set Product Sizes?

  • Ensures the correct locker size is reserved
  • Prevents packages from not fitting
  • Multi-item orders are calculated by stacking dimensions

How to Set Dimensions

  1. Go to the Product Sizes tab
  2. Find your product in the list
  3. Enter Length, Width, and Height (all in inches)
  4. Optionally force a specific locker size from the dropdown
  5. Click Save All Changes

Excluding Products from LockerDrop

Some products may not be suitable for locker pickup (fragile items, oversized goods, etc.). You can exclude specific products:

  1. Go to the Product Sizes tab
  2. Find the product you want to exclude
  3. Check the "Exclude" checkbox in that row
  4. Click Save All Changes

When a customer's cart contains an excluded product, LockerDrop will not appear as a shipping option at checkout.

Tip: Products without dimensions configured are automatically excluded from LockerDrop. Make sure to set dimensions for all products you want available for locker pickup.

Pricing

LockerDrop is free to install. You only pay a small per-order fee each time a customer uses locker pickup at checkout.

How Billing Works

Item Cost
App installation Free
Monthly subscription None
Per locker pickup order $1-$2

Billing is handled through Shopify's built-in billing system. You can view your usage in the Billing tab of the dashboard.

Zero risk: You only pay when customers use locker pickup. No orders = no charges.

Custom Branding

You can customize the pickup success page that customers see after collecting their order.

Branding Options

  • Brand Logo - Display your logo at the top of the success page
  • Custom Colors - Match your brand's primary and secondary colors
  • Success Message - Personalized thank-you message
  • Upsell Products - Show recommended products with links to your store
  • Rebuy Integration - Connect Rebuy for AI-powered recommendations
  • Custom CSS - Advanced styling for complete control

Setting Up Branding

  1. Go to the Branding tab
  2. Upload your logo URL
  3. Choose your brand colors using the color picker
  4. Write a custom success message
  5. Add upsell product IDs (found in Shopify admin)
  6. Click Save Changes
  7. Use the Preview link to see your branded page

Rebuy Integration

If you use Rebuy for product recommendations:

  1. Enable "Rebuy widget" checkbox
  2. Enter your Rebuy Widget ID (found in Rebuy dashboard)
  3. Save changes
  4. Rebuy's AI recommendations will appear on the success page

Troubleshooting Common Issues

"Locker won't open with my link"
  1. Double-check you're at the correct locker location
  2. Make sure you're using the DROP-OFF link (not the customer pickup link)
  3. Try refreshing the page and clicking the link again
  4. Contact Harbor Locker support if the issue persists
"Customer can't find the locker"
  1. Send them the exact address from the order details
  2. Share the locker location name
  3. Harbor lockers are usually near entrances of partner locations
"Package doesn't fit"
  1. Don't drop off if the package won't fit
  2. Contact LockerDrop support to reassign to a larger locker
